## Productify Code Assistant

This custom agent is designed for the Productify workspace in VS Code. It is best used for code changes, development work, and repository-aware problem solving.

### What it does
- Understands `backend/` and `frontend/` directories
- Prioritizes editing existing code files over broad rewrites
- Asks for clarification when task scope is unclear
- Uses VS Code file tools and avoids external terminal or browser actions unless explicitly requested

### When to pick this agent
- You want a programming-focused assistant for Productify
- You need a specialist for backend/frontend feature work in this repo
- You want the assistant to behave as a code editor-aware, workspace-conscious helper
